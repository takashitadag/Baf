# Marvonix Education – verze 2

Produkční statický web připravený pro GitHub Pages / Netlify.

## Co je opravené
- Všechny interní odkazy a assety používají relativní cesty, takže fungují i na GitHub Pages v podadresáři.
- CSS je vložené přímo do HTML jako bezpečný fallback, aby se web nerozpadl při chybě cesty ke stylesheetu.
- Doplněná responzivita, mobilní menu, SEO metadata, Open Graph, manifest, robots, sitemap a bezpečnostní hlavičky pro Netlify.
- Texty jsou formulované jako hotová vzdělávací iniciativa, ne jako příprava.

## Nasazení
Nahraj obsah složky do rootu repozitáře. Neuploaduj nadřazenou složku, ale přímo soubory `index.html`, `style.css`, `programy/`, `kontakt/` atd.
