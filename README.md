# Vzdělávací projekty Martina Sobotky

Profesionální statický web pro prezentaci dvou vzdělávacích programů:

- **Finanční gramotnost v praxi**
- **Práva studentům**

Web je připravený pro nahrání na GitHub a nasazení přes GitHub Pages nebo Netlify.

## Struktura

```text
index.html
style.css
script.js
assets/favicon.svg
```

## Právně bezpečná formulace

Na webu je Marvonix uveden jako značka/týmová podpora, nikoliv jako právnická osoba:

> Vzdělávací projekty založil a provozuje Martin Sobotka. Technickou správu, organizaci a rozvoj projektů podporuje tým Marvonix.

Projekt **Práva studentům** je vymezen jako informační a vzdělávací projekt. Text výslovně uvádí, že nejde o právní poradenství ani právní služby.

## Nasazení na GitHub Pages

1. Vytvoř nový repozitář na GitHubu.
2. Nahraj obsah této složky do repozitáře.
3. V repozitáři otevři **Settings → Pages**.
4. Vyber větev `main` a složku `/root`.
5. Ulož nastavení.

## Nasazení na Netlify

1. Přetáhni celou složku nebo ZIP do Netlify Deploys.
2. Netlify automaticky nasadí statický web.
3. Později můžeš připojit vlastní doménu.
